Glossary of statistical metrics used on CIIPro




Number of compounds tested:

      Number of compounds tested in the assay.

Number of true positives (TP):

      Number of true positives in the assay. A true positive is defined as a compound whose response in the bioassay is active and its in vivo activity is active.

Number of true negatives (TN):

      Number of true negatives in the assay. A true negative is defined as a compound whose response in the bioassay is inactive and in vivo activity is inactive.

Number of false positives (FP):

      Number of false positives in the assay. A false positive is defined as a compound whose response in the bioassay is active and in vivo activity is inactive.

Number of false negatives (FN):

      Number of false negatives in the assay. A false negative is defined as a compound whose response in the bioassay is inactive and in vivo activity is active.

Sensitivity:

      $Sensitivity = \frac{true\thinspace positives}{true\thinspace positives + false\thinspace negatives}$

Specificity:

      $Specificity = \frac{true\thinspace negatives}{true\thinspace negatives + false\thinspace positives}$

Correct Classifcation Rate (CCR):

      $CCR =\frac{Sensitivity + Specificity}{2}$

Positive Predictive Value (PPV):

      $PPV =\frac{true\thinspace positives}{true\thinspace positives + false\thinspace positives}$

Negative Predictive Value (NPV):

      $NPV =\frac{true\thinspace negatives}{true\thinspace negatives + false\thinspace negatives}$

L parameter:

      $L\thinspace parameter = \frac{true\thinspace positives}{true\thinspace positives + false\thinspace negatives} \times \frac{false\thinspace positives + true\thinspace negatives}{false\thinspace positives + 1}$

Coverage:

      $Coverage = \frac{Number\thinspace of\thinspace compounds\thinspace tested}{Number\thinspace of\thinspace compounds\thinspace in\thinspace dataset}$
